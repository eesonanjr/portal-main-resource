# portal-main-resource

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可以在不同场景下直接使用，不绑定特定域名或服务。

## 项目简介

这是一个资源型仓库，主要存放可直接运行的 HTML 文件。页面内容涵盖多种用途，方便快速部署和访问。仓库本身不涉及后端逻辑，所有页面均为静态资源。

## 目录说明

- `pages/`：存放所有 HTML 页面文件，每个文件一个独立页面。
- `assets/`：存放页面所需的公共资源，如样式表、脚本、图片等（如有）。
- `docs/`：存放与页面使用相关的说明文档。
- `index.html`：仓库首页，可快速导航到各个页面。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可直接在浏览器中打开。
- 页面之间无依赖关系，可单独复制使用。
- 文件名尽量反映页面功能，便于查找。
- 部分页面可能包含外部资源引用（如 CDN 库），请确保网络可达。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面。
- 请确保新增页面不包含敏感或违规内容。
- 页面应尽量简洁，避免引入过多外部依赖。
- 提交前请检查页面是否在主流浏览器中正常工作。

## 使用方式

1. 克隆本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的 HTML 文件。
3. 或部署到任意静态文件服务器，即可通过 URL 访问。
